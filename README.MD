# Windows Self Cleaning Library

This library is a single include header to clear recent execution traces

### Supported traces

- UserAssist
- RecentFiles
- RunMRU
- Automatic/CustomDestinations
- RecentDocs
- Prefetch
- USNJournal (Just full deletion, not arbitrary files)
